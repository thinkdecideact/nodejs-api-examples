# An example of how to create a json API with nestjs and typeorm

## Quick start
```
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Database config
The database config is located in [./ormconfig.json](./ormconfig.json).


## Main codes
The main codes are located in
